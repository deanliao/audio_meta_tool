# Tools for Manipulate Audio Files' Metadata

This project contains Python tools to manipulate audio files' metadata in an efficient way.

## Requirements

This tool relies on a Python's package, mutagen, which is for handling audio metadata.
You can find the source code and documentation for Mutagen on its GitHub repository: https://github.com/quodlibet/mutagen

You can use `pip install mutagen` to install it in your Python enviornment.
